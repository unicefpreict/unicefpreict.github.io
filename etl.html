3. Seperate countries into their own CSVs and dataframes

Redundant work, but originally we were going to use 195 CSV's in order to account for population differences between countries within the same region.

4. Add global region column to each country's dataframe

Added Global region Column to each countries dataframe in order to distinguish our data on a global basis
